# Evidence Distillation

Summaries must separate facts from conclusions.

- Lead with retrieved evidence: tool results, files, symbols, docs, or runtime findings.
- Mark inferences as inferences.
- State whether the issue or decision is localized or systemic.
- State the smallest next action that would reduce uncertainty.
- Include residual risk only when it materially affects the recommendation.
- Avoid repeating raw tool output once the salient facts are captured.
